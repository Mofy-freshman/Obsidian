## 1 本地与远程Git仓库同步
1.GitHub新建空仓库（注意不要添加任何文件）
![[c913f58ccf614d4f8a71bae9351fd0b.png|800]]

2.本地新建文件夹，文件夹内新建README.md文件。右键git bash here,按上图步骤操作

3.在本地仓库文件夹内新建Obsidian仓库，或者直接复制到文件夹内

4.执行命令push到remote 
```
git add .
git commit -m ""
git push origin main
```
